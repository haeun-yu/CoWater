🔄 Redis Pub/Sub 채널
채널 패턴 발행자 구독자 설명
platform.report.{platform_id} moth-bridge core, response-agents 선박 보고서 (position, state, metadata)
detect.{agent_id} detection (비활성) analysis (비활성) 탐지 결과 (현재 비활성)
analyze.{agent_id} analysis (비활성) response 분석 결과 (현재 비활성)
respond.{severity} response-agents core, report, learning 경보 생성 (critical, warning, info)
report.{report_id} report-agents core AI 생성 리포트
learn.rule_update.{agent_id} learning-agents detection, response 규칙 개선 제안 (비활성 agent는 무시)
user.{event_type} frontend supervision, learning 사용자 이벤트 (acknowledge, feedback)
system.alert_acknowledge frontend learning 경보 승인 이벤트
platform:state:{platform_id} moth-bridge (캐시) 캐시된 상태 (TTL: 60초)
