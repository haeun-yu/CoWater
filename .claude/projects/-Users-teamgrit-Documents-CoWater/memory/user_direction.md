**방향 변경**: viz-agent는 CoWater의 종속 서비스가 아닌 **독립 오픈소스 프로젝트**로 구성.

**특징**:
- 데이터 인터페이스 표준화 (`DataContract`)
- 플러그인 어댑터 시스템으로 임의 소스 연동 가능
- 도메인별 설정 패키지 (maritime.json, factory.json 등)
- 프론트는 완전히 데이터 포맷에만 의존, 백엔드 무관

**이점**:
- CoWater 변경 시 영향 없음
- 다른 해양/공장/교통/에너지 관제 시스템에 직접 재사용
- 데이터만 맞추면 즉시 시각화 가능
- 오픈소스로 공개 가능한 구조
