# 미션 생명주기 (Mission Lifecycle)

미션이 등록부터 제거까지 거치는 모든 상태 전이와 이벤트

## 상태 다이어그램

```
Registered → Active → Completed
                   ↓
               Cancelled
                   ↓
               Removed
```

## 각 단계 상세

### 1️⃣ Registered (등록)
- **트리거**: 사용자가 새 미션 등록
- **액션**: 미션 객체 생성, 초기 메타데이터 저장
- **다음**: Active로 전환 대기

### 2️⃣ Active (활성)
- **트리거**: 관리자 또는 자동 활성화
- **액션**: 작업 추천 엔진 활성화
- **다음**: 작업 제안 → 승인 → 실행

### 3️⃣ Completed / Cancelled (완료 / 취소)
- **트리거**: 모든 작업 완료 또는 사용자 취소
- **액션**: 타임스탬프 기록, 최종 리포트 생성
- **다음**: Removed로 아카이빙

### 4️⃣ Removed (제거)
- **트리거**: 자동 또는 수동 정리
- **액션**: 데이터 보관, 활성 인덱스에서 제거
- **참고**: 완전 삭제 아님, 감사 목적으로 보관
