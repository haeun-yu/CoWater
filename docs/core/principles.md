# 설계 원칙 (Design Principles)

CoWater 시스템의 핵심 설계 원칙과 책임, 역할을 정의합니다.

## 핵심 원칙

### 1. 명확한 책임 분리 (Single Responsibility)
각 서비스와 모듈은 하나의 책임을 가집니다.

### 2. 공유 이해 (Ubiquitous Language)
모든 팀원이 이해할 수 있는 공통 용어를 사용합니다.

### 3. 도메인 중심 설계
기술 구현이 아닌 비즈니스 도메인이 설계의 중심입니다.
